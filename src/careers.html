About Flowtica

Flowtica helps traditional businesses automate manual workflows using automation platforms, APIs, and AI/LLMs. You’ll work closely with the founder and see the full customer journey from first call to a workflow in production.

What you’ll do

This role blends customer discovery with technical implementation.

  • Run discovery calls, capture requirements, and align delivery plans with the founder.
  • Own delivery: track next steps, communicate clearly, and manage expectations.
  • Build workflows in n8n (or similar), connect systems via REST APIs, and write small JavaScript/Python helpers for data transformations.
  • Use LLMs to parse unstructured text, add logging/monitoring, and support iterations after launch.

The founder is hands-on for mentorship, pairing, and reviews—you’re expected to learn quickly and keep delivery moving.

Must-have qualifications

  • 1–3 years in automation/integration engineering, technical ops/RevOps, internal tools, or junior software development.
  • Experience with an automation tool or scripting (n8n, Zapier, Make/Integromat, or equivalent).
  • Comfortable reading docs and making authenticated REST API requests with JSON payloads.
  • Ability to write basic JavaScript or Python and work with webhooks or event-driven flows.
  • Strong communication: can run customer calls, ask good questions, and write clear summaries.

Nice-to-have

  • n8n experience.
  • Multi-system workflows (CRM + email + Slack/SMS, etc.).
  • Hands-on use of LLMs for extraction/classification and structured outputs.
  • Basic SQL for logging/analytics and tools like Twilio, Slack, SendGrid, HubSpot, Salesforce.
  • Exposure to B2B ops domains like property management, staffing, or distribution.

What we offer

  • ~$50,000 base salary depending on experience, plus stock options.
  • Performance-based raises as you deliver impact and take on more ownership.
  • On-site in Kingston, ON working closely with the founder.
  • Ownership of end-to-end implementations with a fast learning curve in automation and AI.
