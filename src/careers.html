About Flowtica

Flowtica helps traditional businesses (distributors, property managers, staffing agencies, etc.) automate messy, manual workflows using automation platforms, APIs, and AI/LLMs.

We are early-stage and hands-on. You will work closely with the founder and see the full customer journey — from first call to a workflow running in production.

What You’ll Do

Customer ownership

  • Run customer discovery calls to understand workflows, constraints, and success metrics
  • Capture requirements and turn conversations into clear notes, scope, and acceptance criteria
  • Lead internal scoping with the founder to align on technical approach and delivery plan
  • Own delivery end-to-end: track next steps, drive progress, communicate clearly, manage expectations

Implementation (with mentorship)

  • Build workflows in an automation platform (currently n8n; similar tools relevant)
  • Connect systems via REST APIs (CRMs, ERPs, PMS, ATS, email/SMS, Slack, Google/Microsoft 365)
  • Write small JavaScript/Python helpers to transform, validate, and route data
  • Use LLMs to parse and classify unstructured text (emails, forms, messages)
  • Add logging, basic monitoring, and simple runbooks
  • Support post-launch iterations based on real usage and customer feedback

Must-Have Qualifications

  • 1–3 years relevant experience (automation/integration engineering, technical ops/RevOps, internal tools, or junior software dev)
  • Hands-on experience with at least one automation tool (n8n, Zapier, Make, Integromat, or equivalent)
  • Comfortable with REST APIs and JSON (auth, GET/POST, reading docs)
  • Ability to write basic JavaScript or Python (loops, conditionals, simple helpers)
  • Familiarity with webhooks or event-driven flows
  • Strong communication skills: can run customer calls and write clear summaries

Nice-to-Have

  • n8n experience
  • Multi-system workflows (CRM + email + Slack/SMS, etc.)
  • Hands-on LLM usage for extraction/classification
  • Basic SQL (Postgres/MySQL)
  • Experience with Twilio, Slack, SendGrid, HubSpot, Salesforce
  • Exposure to B2B ops domains (property management, staffing, distribution)

What We Offer

  • ~$50,000 base salary depending on experience
  • Stock options and performance-based raises
  • On-site role in Kingston, ON working closely with the founder
  • Ownership of full implementations, not just small tickets
  • Fast learning curve in automation, integrations, and practical AI for real businesses
