What you’ll do

Customer ownership

  • Run discovery calls to understand workflows, constraints, and success metrics.
  • Capture requirements and turn conversations into clear notes, scope, and acceptance criteria.
  • Lead internal scoping with the founder to align on the technical approach and delivery plan.
  • Own delivery end-to-end: track next steps, drive progress, communicate clearly, and manage expectations.

Implementation (you build and ship with mentorship)

  • Build workflows in an automation platform (currently n8n; similar tools relevant).
  • Connect systems via REST APIs (CRMs, ERPs, PMS, ATS, email/SMS, Slack, Google/Microsoft 365).
  • Write small JavaScript/Python helpers to transform, validate, and route data.
  • Use LLMs to parse and classify unstructured text (emails, forms, messages).
  • Add logging, basic monitoring, and simple runbooks so workflows are reliable and debuggable.
  • Support iterations after launch based on real usage and customer feedback.

About Flowtica

Flowtica helps traditional businesses (distributors, property managers, staffing agencies, etc.) automate messy, manual workflows using automation platforms, APIs, and AI/LLMs. We’re early-stage and hands-on. You’ll work closely with the founder and see the full customer journey from first call to a workflow running in production.

Must-have qualifications

  • 1–3 years of relevant experience (automation/integration engineering, technical ops/RevOps, internal tools, or junior software dev).
  • Practical experience with at least one automation/integration tool or approach (n8n, Zapier, Make, Integromat, or scripting).
  • Comfortable with REST APIs and JSON, including auth such as API keys or bearer tokens.
  • Ability to write basic JavaScript or Python (arrays/objects, loops, conditionals, simple helpers).
  • Familiarity with webhooks or event-driven flows, or ability to pick it up quickly.
  • Strong communication: can run a customer call, ask good questions, and write clean summaries.

Nice-to-have

  • n8n experience specifically.
  • Workflows spanning multiple systems (CRM + email + Slack/SMS, etc.).
  • Hands-on use of LLMs for extraction/classification and structured outputs.
  • Basic SQL (Postgres/MySQL) for logging/analytics.
  • Experience with tools like Twilio, Slack, SendGrid, HubSpot, Salesforce.
  • Exposure to B2B ops domains like property management, staffing, or distribution.
